Was ist Tailscale?

          
Tailscale ist ein VPN-Dienst, der deine Geraete (Pi, Handy, Laptop) ueber das Internet sicher verbindet – als waeren sie im selben Netzwerk. Verschluesselt, kein Portforwarding noetig, kostenlos fuer bis zu 3 Nutzer.


          
1. Tailscale auf dem Pi

          
Tailscale ist auf diesem Pi bereits installiert. Klicke oben auf Einschalten, um es zu starten.


          
2. Erstmalig anmelden

          
Beim ersten Start erscheint ein Anmelde-Link. Diesen im Browser oeffnen, mit Google, Microsoft oder GitHub anmelden. Der Pi wird dann deinem Tailscale-Netzwerk hinzugefuegt.


          
3. Tailscale auf deinem Geraet

          
Installiere Tailscale auf deinem Handy oder Laptop (tailscale.com/download) und melde dich mit dem gleichen Konto an.


          
4. Dashboard aufrufen

          
Sobald beide Geraete verbunden sind, ist das Dashboard ueber die Tailscale-IP erreichbar: http://<Tailscale-IP>. Die IP wird oben angezeigt.
